Starfield Creations Mod Manager

Effortlessly Manage Your Starfield Mods

Features Usage Support

Key Features

  • Manage Creations mods for Starfield with ease.

  • Check and repair corrupted ContentCatalog.txt files.

  • Reorder mods using drag-and-drop or keyboard shortcuts.

  • Enable or disable mods without removing files.

  • Create and switch between mod profiles for different save games.

  • Backup and restore your original mod configurations.

  • Dark mode support for better usability.

How to Use

Follow these steps to get started with the Starfield Creations Mod Manager:

  1. Download and install the application.

  2. Launch the tool and use the intuitive interface to manage your mods.

  3. Use the "Fix Catalog" feature to repair any issues with your ContentCatalog.txt file.

  4. Organize your mods by dragging and dropping them in the desired order.

  5. Switch between different mod profiles for separate save files.
